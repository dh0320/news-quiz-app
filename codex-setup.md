# Codex セットアップ手順書

OpenAI Codex (ChatGPT Pro) を使って、毎日自動でニュースクイズエピソードを生成する設定手順。

## 1. リポジトリ接続

1. [ChatGPT](https://chatgpt.com) にログイン
2. 左サイドバーから **Codex** を選択
3. **Connect repository** をクリック
4. GitHub アカウントを連携し、`news-quiz-app` リポジトリを選択
5. リポジトリが接続されると、`AGENTS.md` が自動で認識される

## 2. スケジュールタスクの設定

**1ジャンル = 1タスク** で5つのスケジュールを設定する。
これにより、各ジャンルに集中したニュース検索と高品質なエピソード生成が可能になる。

### タスク1: 政治・政策 (09:00 JST)

**プロンプト:**
```
本日のニュースクイズエピソードを生成してください。

ジャンル: politics_policy（政治・政策）
ファイル番号: 01

AGENTS.md の「エピソード生成タスク」の手順に従ってください。
Web検索で今日の日本の政治・政策ニュースを1つ選び、エピソードJSONを生成、バリデーションを通してからコミットしてください。
```

### スケジュール設定のポイント

- **15分間隔** で設定することで、コミットの競合を避ける
- 各タスクは独立しているので、1つが失敗しても他に影響しない
- Codex UIの **Schedule** 機能で曜日・時間を設定可能
- 平日のみ（月〜金）にすることも可能

## 3. 手動実行

スケジュール外で手動生成したい場合:

1. Codex画面で **New task** をクリック
2. 上記のプロンプトのいずれかを貼り付けて実行
3. または全ジャンル一括で:
   ```
   本日のニュースクイズエピソードを全5ジャンル分生成してください。
   AGENTS.md の手順に従い、1ジャンルずつ順に生成・バリデーション・コミットしてください。
   ```

## 4. バックアップ: Claude Code CLI

Codex が動作しない場合、Claude Code CLI で手動生成可能:

```bash
# 全5ジャンル一括
bash scripts/generate-episode.sh

# 特定日付で生成
bash scripts/generate-episode.sh 2026-03-07

# 特定ジャンルのみ
bash scripts/generate-episode.sh 2026-03-07 politics_policy 01
```

## 5. トラブルシューティング

### バリデーションエラーが出る
- `npm run validate` のエラーメッセージを確認
- `episodeId` とファイル名の不一致が多い原因
- Codexのタスク結果のログで詳細を確認

### コミットの競合
- 複数タスクが同時に実行されると `git push` で競合する場合がある
- スケジュールの間隔を広げる（15分→30分）ことで回避可能

### ニュースが見つからない
- 特定ジャンル（career_workstyle等）は日によってニュースが少ない
- Codexが「適切なニュースが見つからなかった」と報告した場合は、翌日に持ち越し

## 6. 自動デプロイの確認

1. Codexがコミット & プッシュに成功すると
2. Vercel が自動でビルドを開始
3. 数分後に本番サイトに新エピソードが反映される
4. Vercel ダッシュボードでデプロイ状況を確認可能
